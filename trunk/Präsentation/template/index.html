Schwerpunkt: Android 4-API

ActionBar

Vorteile:

  • Navigation durch App
  • Darstellung von Menü-Einträgen
  • Wegfall einer Hardware-Taste

Nachteile:

  • Platzverbrauch
  • Geräte mit Menü-Taste

Fragments

Vorteile:

  • Ausnutzung großer Displays
  • Modularisierung
  • Lifecycles

Nachteile:

  • Lifecycles

ViewPager

Vorteile:

  • Wisch-Gesten

Nachteile:

  • Umgang mit Fragments

Contextual ActionBar

Vorteile:

  • Mehrfachselektierung
  • Neue ActionBar

Nachteile:

  • Gedrückthalten zum Auswählen
